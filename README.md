## Prerequisite

- Node.js version >= 7
- [Yarn](https://yarnpkg.com)
- [Bower](https://bower.io)

## Installation

```shell
yarn
```

and 

```shell
bower install
```

## Usage

### Production Mode

```shell
NODE_ENV=production node .
```

### Development Mode

In order to develop locally, this mode will run without SSL

```shell
node .
```

App will listen on port 9090 by default

## Test

```shell
yarn run test
```
